一、SDK简述

1. SDK目录说明

SDK包含四个目录，每个目录包含的文件如下：
1. bin目录-------包含demo二进制
2. include目录---包含SDK头文件，API具体参数及说明
3. lib目录-------包含SDK的.lib和.dll文件
4. src目录-------包含demo源码

2. SDK使用说明

1. 特殊用途的API，这些一般不需要关注和调用。
InitPlatform
SetBusinessType
SetVideoCaptureFactory
SetPlatformInfo
SetUseTestEnv

2. 在InitSDK之前可以调用的API：
SetLogLevel(建议InitSDK之前调用)   
SetCallback(建议InitSDK之前调用)
GetSDKVersion
GetAudioDeviceList
GetVideoDeviceList

3. 大多数API都需要在InitSDK之后才能调用，其它API必须在InitSDK之后根据需要调用。

4. API都是异步的，不会堵塞调用线程，要求都在主线程调用。
5. 所有的字符串参数都要求是 utf8 格式。

0xe3,0xdd,0xdb,0xe0,0xf1,0x74,0x36,0xd };

// 初始化sdk

ZEGO::AV::SetPreviewView(hView);
ZEGO::AV::StartPreview();

ZEGO::AV::StartPublish(AvRoomApp::Utility::WStringToUTF8(m_strTitle).c_str(), AvRoomApp::Utility::WStringToUTF8(m_strStreamID).c_str());
}

十、错误码说明

在Publish和Play的回调中出现以下错误码 说明：

错误码 说明
AVStateEnd = 1 直播正常停止
TempBroken = 2 直播临时中断
FatalError = 3 直播遇到严重的问题
CreateStreamError=4 创建直播流失败
FetchStreamError=5 获取流信息失败
NoStreamError=6 无流信息
MediaServerNetWorkError=7 媒体服务器连接失败
DNSResolveError=8 DNS 解释失败
